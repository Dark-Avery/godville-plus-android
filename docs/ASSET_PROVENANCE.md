# Asset Provenance

## Application icon

The application icon is a project-authored pixel-style `GV+` monogram. Its
vector sources are stored in `app/src/main/res/drawable`, with adaptive icon
definitions in `app/src/main/res/mipmap-anydpi-v26`.

The icon was designed specifically for this project. It is not copied from
Godville, Erinome, the Godville page header, or their halo and torch artwork.

## Erinome assets

Assets below `app/src/main/assets/erinome` come from the bundled Erinome Godville
UI+ 1.1.39.8 snapshot and are covered by the Erinome License. Android-specific
source changes are listed in `THIRD_PARTY_NOTICES.md`.
